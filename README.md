# Mini-shell
Este proyecto es un mini shell personalizado desarrollado en Python que permite al usuario interactuar con el sistema operativo mediante comandos propios. Ofrece funcionalidades como consulta de fecha y hora, visualización y gestión de procesos, monitoreo del sistema (CPU, memoria y disco), exploración de archivos, información de red y limpieza de pantalla.

El shell implementa un bucle interactivo que interpreta comandos ingresados por el usuario, simulando el comportamiento de una terminal básica. Además, utiliza librerías como psutil, os, platform y socket para acceder a información del sistema y ejecutar tareas administrativas, proporcionando una herramienta práctica para la gestión y monitoreo del entorno del sistema.
